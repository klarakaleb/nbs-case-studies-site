Use the buttons below to submit a new case study, suggest a source, or review pending proposals from the AI agent. All review happens in a shared Google Sheet — no GitHub account needed.

How it works

  1. You submit (or the agent finds) something needing attention.
  2. The agent does the legwork — fetches sources, checks links, archives them to the Wayback Machine, drafts a proposal.
  3. The proposal appears in the team Sheet under Pending review.
  4. A team member picks Approve / Reject in the decision column.
  5. The agent reads decisions overnight and applies approved changes to the case study database.

Humans approve every change. The agent does the legwork; the team does the judgement.
